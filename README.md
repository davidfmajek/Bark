## BARK - UMBC Campus Dining Reviews

## Problem Statement
UMBC students lack a centralized platform for reviewing campus dining locations.

## Target Users
- Current UMBC students
- Incoming freshmen

## Tech Stack 
Frontend: React
Backend: Flask
Database: PostgreSQL
API: Google Maps API
Visualization: Chart.js

## Core Features
- CRUD Reviews
- Location search
- Ratings visualization
- External API integration
